阅读该百科需要知晓的事项。

## 命令行调用

命令行调用时，最后两个参数通常都是输入/输出文件。

本文会在命令行调用例子的行末使用 `\` 符号来表示下一行的内容是连续的 ——
这在 unix 终端环境下是合法的。

在 Windows 下你可能需要将该符号替换为 `^` (cmd) 或 <code>\`</code> (PowerShell)
符号，或直接改为空格连接参数即可。

## 伪代码

项目包含了部分算法的伪代码描述。

除非特殊注明，否则：

- `array[index]` 表示 `array[index % len(array)]`
- `i..j` 表示区间 `[i, j)` 的所有整值。步间为 `+1`。
- `pack/unpack` 函数用于表示整数到字节组的变化关系。
  - 与 Python 的 [struct 文档][py_struct]描述的行为一致。

[py_struct]: https://docs.python.org/3/library/struct.html
