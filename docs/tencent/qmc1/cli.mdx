## QMCv1

支持 `key58` 或 `key128` 格式的密钥。

```sh title="使用测试密钥来解密文件"
cargo run --release qmc1 --static-key "@qmc1_key58.bin" \
  "sample/test_qmc1.qmcogg" "decrypted.ogg"

cargo run --release qmc1 --static-key "@qmc1_key128.bin" \
  "sample/test_qmc1.qmcogg" "decrypted.ogg"
```

### 参数：

- `--static-key` - 流解码密钥。可以是 `58key` 或 `128key` 的密钥。

之后附加两个参数，分别为 `输入文件路径` 与 `输出文件路径`。
